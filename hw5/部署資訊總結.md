## 🔧 部署後設置（5 分鐘）

### 1. 更新 NEXTAUTH_URL
```
Vercel → Settings → Environment Variables
NEXTAUTH_URL = https://您的app.vercel.app
Save → Redeploy
```

### 2. 運行資料庫遷移
```bash
cd /Users/joy.lin/Desktop/wp1141/hw5

# 從 Vercel 取得環境變數
vercel env pull .env.production

# 運行遷移
npx prisma migrate deploy
```

或手動：
```bash
export DATABASE_URL="postgresql://從Vercel複製的URL"
npx prisma migrate deploy
```

### 3. 更新 OAuth 回調 URL

**Google**:
```
https://console.cloud.google.com/
→ 添加: https://您的app.vercel.app/api/auth/callback/google
```

**GitHub**:
```
https://github.com/settings/developers
→ 更新: https://您的app.vercel.app/api/auth/callback/github
```
