## 啟動應用程式

### 1. 啟動後端
```bash
cd backend
npm install  # 如果還沒安裝
npm run dev
```
後端將運行在 http://localhost:3000

### 2. 啟動前端
```bash
cd frontend
npm install  # 如果還沒安裝
npm run dev
```
前端將運行在 http://localhost:5173
