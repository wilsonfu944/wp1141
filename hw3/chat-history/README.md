# Chat History 說明

## 📝 請將 Cursor Chat History 放置於此

### 如何下載 Chat History

1. 在 Cursor 中開啟 Chat 面板
2. 點擊右上角的選單圖示（⋮）
3. 選擇 "Download chat history"
4. 儲存 .md 檔案到此資料夾

### 檔案大小限制

如果所有 chat history .md 檔案總和 **超過 100KB**：

請使用 Cursor 或其他 AI 工具處理檔案，**只保留你的 prompts**（使用者提問），移除 AI 的回應內容。

### 處理 Prompt 範例

```
請幫我處理這個 chat history .md 檔案，只保留我的 prompts（使用者提問），
移除所有 AI 回應的內容。保持原有的格式和時間戳記。
```

### 檢查檔案大小

```bash
# 在此資料夾中執行
du -sh *.md

# 或查看總大小
du -sh .
```

### 注意事項

- ✅ 請保留所有 .md 檔案的原始檔名
- ✅ 如果有多個 chat，請都放在此資料夾
- ✅ 確保檔案總大小不超過 100KB
- ❌ 不要上傳二進位檔案或其他格式

---

**重要：繳交前請確認此資料夾內已有 chat history .md 檔案！**
