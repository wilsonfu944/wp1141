## 使用建議

### 測試場景
1. **商品瀏覽**：利用多樣的分類與價格區間測試篩選功能
2. **搜尋功能**：測試品牌名稱、商品特徵關鍵字搜尋
3. **購物車**：測試不同價格與數量的商品組合
4. **庫存管理**：測試缺貨、低庫存、高庫存等不同狀態
5. **排序功能**：利用價格與評分差異測試排序準確性

### 擴展建議
如需新增商品，建議：
1. 保持分類比例平衡
2. 價格範圍符合該分類的市場行情
3. 評分在 4.0-5.0 之間較為真實
4. 特徵數量建議 3-5 個
5. 描述長度建議 20-50 字

### 自訂修改
編輯 `public/data/products.csv` 檔案即可：
- 修改價格以符合不同市場
- 調整庫存數量模擬不同狀態
- 更新圖片 URL 使用自有圖片
- 新增或修改商品特徵
- 調整評分以符合需求

## 注意事項

1. **圖片載入**：需要網路連線才能從 Unsplash 載入圖片
2. **品牌商標**：品牌名稱僅供學習使用，商業用途請注意版權
3. **價格時效**：價格為 2024-2025 年參考值，實際價格可能變動
4. **庫存模擬**：庫存數量為模擬資料，非真實庫存
5. **評分來源**：評分為參考市場平均，非特定平台資料
