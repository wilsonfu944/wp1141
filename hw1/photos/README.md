## 照片規格

### 封面照片 (osaka1.jpg, kyoto1.jpg, tokyo1.jpg)
- **用途**：顯示在分類卡片上
- **建議尺寸**：300x200 像素
- **檔案大小**：< 500KB
- **格式**：JPG 或 PNG

### 相簿照片 (osaka2-9.jpg, kyoto2-9.jpg, tokyo2-9.jpg)
- **用途**：顯示在相簿網格中
- **建議尺寸**：250x200 像素
- **檔案大小**：< 300KB
- **格式**：JPG 或 PNG

## 如何添加照片

### 1. 準備照片
- 選擇您想要展示的旅行照片
- 使用圖片編輯軟體調整尺寸
- 壓縮檔案以減少載入時間

### 2. 重新命名
- 按照上述命名規則重新命名檔案
- 確保檔案名稱完全一致（包括大小寫）

### 3. 放置檔案
- 將照片放入對應的資料夾
- 確保檔案路徑正確

### 4. 測試
- 重新整理網頁
- 檢查照片是否正常顯示
- 測試相簿功能

## 注意事項

- 如果照片無法載入，會自動顯示 placeholder 圖片
- 建議使用 JPG 格式以獲得更好的壓縮效果
- 保持照片的長寬比以避免變形
- 定期備份原始照片

## 故障排除

### 照片不顯示
1. 檢查檔案是否存在
2. 確認檔案名稱是否正確
3. 檢查檔案權限

### 照片載入緩慢
1. 壓縮照片檔案
2. 調整照片尺寸
3. 使用較低的品質設定
