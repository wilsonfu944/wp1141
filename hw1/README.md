# 個人首頁 

這是一個使用 HTML、CSS、TypeScript 建立的個人首頁靜態網頁，展現個人特色與興趣專題。

## 網頁設計概念

### 整體設計理念
本網頁採用**現代簡約風格**，以**個人品牌**為核心，結合**互動式元素**和**視覺層次**，創造出既專業又親切的個人展示平台。

### 色彩設計
- **主色調**：深藍色系 (#2c3e50) - 傳達專業與穩重
- **輔助色**：漸層色彩 - 增加視覺豐富度
- **強調色**：亮色系按鈕 - 引導使用者互動
- **背景**：柔和漸層 - 營造舒適的視覺環境

### 版面設計
- **首頁區塊**：全螢幕英雄區塊，搭配動態背景
- **關於我**：個人介紹與特色展示
- **興趣專題**：音樂卡片網格佈局，支援互動播放
- **旅行相簿**：分類相簿展示，支援相簿切換功能

### 互動設計
- **平滑滾動**：導覽列點擊時平滑滾動到對應區塊
- **音樂播放**：卡片懸停效果與播放控制
- **相簿瀏覽**：點擊展開相簿，支援關閉功能
- **響應式互動**：適配不同裝置的觸控操作

## 功能特色

- 🏠 **個人首頁**：歡迎詞和動態背景效果
- 👤 **關於我**：個人介紹和特色展示
- 🎵 **音樂專題**：互動式音樂卡片，支援播放/暫停功能
- 📸 **旅行相簿**：分類相簿展示（大阪、京都、東京）
- 📱 **響應式設計**：適配各種螢幕尺寸
- 🎨 **現代化 UI**：簡潔美觀的設計風格
- ⚡ **JavaScript**：原生 JavaScript 開發
- 🎭 **動畫效果**：平滑的過渡動畫和互動回饋

## 快速開始

### 方法一：使用編譯腳本（推薦）

```bash
# 執行編譯腳本
./compile.sh
```

### 方法二：手動編譯

```bash
# 安裝 TypeScript（如果尚未安裝）
npm install -g typescript

# 編譯 TypeScript
npx tsc

# 啟動本地伺服器
python3 -m http.server 8000
```

### 方法三：使用 npm scripts

```bash
# 安裝依賴
npm install

# 編譯並啟動開發伺服器
npm run dev

# 或只啟動伺服器
npm start
```

## 瀏覽網頁

編譯完成後，在瀏覽器中開啟：
- http://localhost:8000

## 功能說明

### 導覽功能
- 點擊導覽列項目可平滑滾動到對應區塊
- 點擊 "Learn More" 按鈕滾動到關於我區塊

### 音樂播放功能
- 滑鼠懸停在音樂卡片上會顯示播放按鈕
- 點擊播放按鈕或卡片可播放/暫停音樂
- 支援多首歌曲切換播放

### 響應式設計
- 桌面版：完整功能展示
- 平板版：調整卡片佈局
- 手機版：單欄式佈局

## 自訂內容

### 修改個人資訊
編輯 `index.html` 中的以下內容：
- 姓名：`陳劭瑋`
- 自我介紹文字
- 音樂清單

### 更換圖片
- 背景圖片：修改 CSS 中的 `background-image` URL
- 個人照片：修改 `about-image` 中的 `src` 屬性
- 專輯封面：修改各音樂卡片的 `src` 屬性

### 添加音樂檔案
1. 建立 `music/` 資料夾
2. 將音樂檔案命名為 `song1.mp3`, `song2.mp3` 等
3. 確保檔案格式為 MP3
