## 👹 怪物系統設計

### 怪物類型
遊戲包含兩種不同類型的怪物，增加遊戲的挑戰性：

#### 飛行怪物 (Flying Monsters)
- **外觀**：紅色圓形，有浮動動畫效果
- **行為**：水平移動 + 上下浮動
- **威脅**：中等，需要預判其移動軌跡

#### 行走怪物 (Walking Monsters)
- **外觀**：棕色方形，有重力效果
- **行為**：水平移動 + 受重力影響
- **威脅**：較高，移動模式更不可預測

### 怪物生成機制
- **觸發條件**：分數達到 2000 分時開始生成
- **生成頻率**：每 3 秒有 0.5% 的機率生成新怪物
- **數量限制**：同時最多存在 2 隻怪物
- **清理機制**：怪物離開螢幕範圍或距離玩家過遠時會被清理

### 碰撞檢測
- **精確檢測**：使用矩形碰撞檢測算法
- **即時反應**：碰撞時立即結束遊戲
- **視覺反饋**：碰撞後有短暫的延遲才顯示遊戲結束畫面

## 🎮 遊戲玩法說明

### 基本操作
- **左右移動**：使用 ← → 箭頭鍵或 A/D 鍵
- **邊界穿越**：角色可以從螢幕一邊穿越到另一邊
- **跳躍機制**：踏上平台時自動跳躍
- **暫停功能**：按 ESC 鍵可以暫停遊戲

### 遊戲目標
- **主要目標**：控制角色不斷向上跳躍，踩踏平台獲得更高分數
- **次要目標**：躲避怪物，挑戰更高難度
- **終極目標**：創造新的最高分記錄

### 分數系統
- **計算方式**：基於角色相對於地面的高度
- **分數公式**：`Math.floor((groundLevel - playerY) / 10)`
- **最高分記錄**：自動保存玩家的最高分數
- **排行榜**：支持多個玩家的分數比較

### 物理引擎
- **重力系統**：角色受重力影響向下墜落
- **跳躍力**：踏上平台時獲得向上的跳躍力
- **摩擦力**：水平移動時有摩擦力減速
- **碰撞檢測**：精確的平台和怪物碰撞檢測
