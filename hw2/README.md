# 🐍 貪食蛇遊戲

這是一個使用 React + TypeScript 建立的現代化貪食蛇遊戲專案，具有美觀的 UI 設計和多食物系統。

## ✨ 功能特色

### 🎮 遊戲機制
- **多食物系統**：場上同時存在 3 個食物，增加遊戲挑戰性
- **固定位置**：食物位置固定不變，只在被吃掉後生成新食物
- **自動移動**：蛇每 200ms 自動移動一次
- **方向控制**：使用方向鍵控制蛇的移動方向
- **防反向**：蛇不能立即反向移動，避免意外自撞
- **關卡模式**：三個不同難度的關卡，每個關卡都有獨特的障礙物設計

### 🎨 視覺設計
- **漸層背景**：淺藍到紫色的夢幻漸層背景
- **棋盤格**：Canvas 上的淺灰色格線，提供清晰的視覺參考
- **美化蛇身**：綠色漸層、圓角設計、陰影效果
- **美化食物**：紅色圓角方塊，帶有縮放消失動畫
- **分數卡片**：右上角顯示當前分數和最高分
- **Game Over 畫面**：半透明黑底 + 白色大字，美觀的重新開始按鈕

### 🔤 字體設計
- **Press Start 2P**：使用 Google Fonts 像素風格字體，營造復古遊戲感

## 🚀 安裝與執行

### 系統需求
- Node.js 16.0 或更高版本
- npm 或 yarn 套件管理器

### 安裝步驟

1. **安裝依賴套件**：
```bash
npm install
```

2. **啟動開發伺服器**：
```bash
npm start
```

3. **開啟遊戲**：
在瀏覽器中訪問 http://localhost:3000

### 生產環境建置
```bash
npm run build
```

## 🎨 設計特色

- **毛玻璃效果**：使用 `backdrop-filter: blur()` 創造現代感

- **圓角設計**：所有元素都使用圓角，更加柔和
